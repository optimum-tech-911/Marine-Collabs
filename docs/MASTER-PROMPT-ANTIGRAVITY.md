# Prompt de continuation pour Antigravity / Gemini / Claude Code

Tu interviens sur la plateforme française RNI — Régie Nautique d’Influence.

Commence par lire `AGENTS.md`, `README.md`, les fichiers `docs/`, puis le plan de recherche dans `research/`.

Ta mission est d’améliorer cette base sans la reconstruire :

- conserver les routes, la shortlist et le campaign builder ;
- renforcer les cartes et profils comme media-kits B2B ;
- remplacer progressivement les placeholders générés par les médias approuvés ;
- centraliser toute nouvelle donnée ;
- améliorer les pages Solutions et Pour les marques dans la même direction visuelle ;
- connecter le CTA « Choisir un créneau » à l’URL de réservation définie dans `src/config/brand.ts` ;
- maintenir les statuts de preuve des métriques ;
- ne jamais inventer une audience, une performance, une collaboration ou une vente ;
- tester desktop, tablette, mobile, clavier et reduced-motion ;
- exécuter `npm run check` et corriger toutes les erreurs avant de terminer.

Le profil Mégane est la référence de qualité pour le système complet. Une fois parfait, réutiliser les mêmes composants pour les onze autres profils au lieu de hardcoder des pages différentes.
