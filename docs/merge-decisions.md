# Merge decisions

- The root is the unified deployable application; `RNI-Regie-Nautique-Influence-V5/` is retained only as a non-deployed source reference.
- V5 components were promoted instead of maintaining two routers, two global stylesheets or two creator datasets.
- Locale paths are canonical. The root is only a temporary locale-entry URL; old unprefixed routes receive temporary controlled redirects.
- Browser language is preferred over country. Country is a conservative final fallback and does not override a localized URL or saved choice.
- Creator identifiers, handles, metrics, sources and media provenance remain shared; editorial copy is the localization boundary.
